У mixed_parser есть параметр:
    --table [0 или 1] - 1 (стандартное значение) - записи берутся уже из готовой базы
                      - 0 -  то качает файл и читает данные из него целиком без шагов

Парсер обрабатывает 10.000 доменов примерно за 900 секунд.
5.000.000 rows / (10.000 pages / 900 secs) / 60 sec / 60 min / 24 hors = 5.6 days
